Ordered List

  1. Elements in order : Items are sorted in specific order.
  2. Can have duplicates :You can have repeated elements.
  3. Access by index : You can access elements by their position.
  4. Add or remove : You can add or remove elements as needed.
  5. Maintain order : The order of elements is maintained.
  6. Flexible :You can modify the list as needed.
